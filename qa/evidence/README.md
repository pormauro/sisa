# Evidencias QA

Esta carpeta guarda corridas manuales auditables del Milestone 5.

## Regla de uso

- crear un archivo nuevo por corrida a partir de `qa/MULTI_DEVICE_EVIDENCE_TEMPLATE.md`
- nombre sugerido: `YYYY-MM-DD-escenario-dispositivo.md`
- no sobrescribir evidencias viejas; abrir una nueva corrida aunque sea una repeticion

## Minimo esperado

- escenario ejecutado
- UUIDs y `device_id` usados
- resultado esperado vs real
- evidencia adjunta o referencia a capturas/logs
- decision final: `pass`, `fail` o `bloqueado`
